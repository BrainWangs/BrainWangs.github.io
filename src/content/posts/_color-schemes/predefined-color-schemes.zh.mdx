AstroPaper 内置了一系列预定义配色方案，可用于自定义主题外观。每个方案都定义了一套完整的 CSS 自定义属性（变量），涵盖浅色和深色模式。

## 目录

## 快速开始

要应用预定义配色方案，请将 CSS 变量定义复制到你的主题配置中。详细设置说明请参阅[配色方案配置指南](https://astro-paper.pages.dev/posts/customizing-astropaper-theme-color-schemes/)。

## CSS 变量参考

所有配色方案都使用以下 CSS 自定义属性：

<ResponsiveTable variant="striped-minimal">

| 变量                   | 用途                           |
| ---------------------- | ------------------------------ |
| `--background`         | 主背景色                       |
| `--foreground`         | 主文本色                       |
| `--accent`             | 强调/交互元素（链接、按钮、高亮） |
| `--accent-foreground`  | 强调色背景上的文本颜色            |
| `--muted`              | 次要区域的辅助背景色              |
| `--muted-foreground`   | 次要内容的文本颜色               |
| `--border`             | 边框和分隔线颜色                 |

</ResponsiveTable>

## 浅色方案

浅色配色方案使用 CSS 选择器 `:root` 和 `[data-theme="light"]` 来定义。

### Paper Light

AstroPaper 默认浅色主题。

![Paper Light](assets/paper-light.png)

```css
:root,
[data-theme="light"] {
  --background: #fdfdfd;
  --foreground: #282728;
  --accent: #006cac;
  --accent-foreground: #ffffff;
  --muted: #e6e6e6;
  --muted-foreground: #6b7280;
  --border: #ece9e9;
}
```

### Kha-Yan

![Kha-Yan](assets/kha-yan.png)

  --background: #fefaec;

  --accent-foreground: #fefaec;

## 深色方案

深色配色方案使用 CSS 选择器 `[data-theme="dark"]` 来定义。

### Paper Dark

AstroPaper 原始深色主题，以青色作为强调色。

![AstroPaper 1 default dark theme](https://user-images.githubusercontent.com/53733092/215769153-13b0ad8d-5ba2-44b1-af06-e5ae61293f62.png)

```css
[data-theme="dark"] {
  --background: #2f3741;
  --foreground: #e6e6e6;
  --accent: #1ad9d9;
  --accent-foreground: #0d2b2b;
  --muted: #596b81;
  --muted-foreground: #8faabb;
  --border: #3b4655;
}
```
